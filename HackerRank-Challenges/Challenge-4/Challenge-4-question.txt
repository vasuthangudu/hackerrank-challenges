Create a program that takes the number of wins, draws and losses and calculates the number of points a football team has obtained so far.

wins get 3 points
draws get 1 points
losses get 0 points
Input Format

number of wins, draws and losses

Constraints

values should not be negative

Output Format

number of points
Ex:
4 2 1 ===> 4x3 + 2x1 + 1x0 ===> 12 + 2 + 0 ===> 14
3 4 2 ===> 3x3 + 4x1 + 2x0 ===> 9 + 4 + 0 ====> 13

Sample Input 0

4 2 1
Sample Output 0

14
